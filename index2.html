¿Que es la programcion?

La programación es el arte de darle instrucciones a una computadora para que realice tareas específicas. Estas instrucciones se escriben en lenguajes que las máquinas pueden entender, como Python, Java o C++, y permiten desde crear una app para ordenar comida hasta desarrollar inteligencia artificial o sistemas que controlan un satélite.
¿Para qué sirve?
· Automatiza tareas repetitivas para ahorrar tiempo y esfuerzo.

· Permite desarrollar aplicaciones, videojuegos, sitios web y software.

· Es fundamental para la tecnología moderna: desde los autos inteligentes hasta la banca en línea.

· Impulsa la innovación en medicina, educación, ciencia y muchas otras áreas.
